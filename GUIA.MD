A DOCUMENTAÇÃO DA API ESTÁ NO http://localhost:4000/api-docs

Existem algumas dependências como citado no README, então ao rodar o npm install tem que dar o seguinte comando
npm install --legacy-peer-deps

Após isso realizar um cd server para entrar no diretório do servidor
E realizar o código acima novamente
npm install --legacy-peer-deps

E após isso só executar o npm start, que ele já vai rodar o servidor e ai é possivel abrir o mvc e api também.
